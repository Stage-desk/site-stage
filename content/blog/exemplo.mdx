# Bem-vindo ao Blog da Stage

Este é um **artigo de exemplo** escrito em *Markdown* (especificamente MDX) para demonstrar as capacidades do nosso novo blog.

## O poder do Markdown

O Markdown permite formatar o texto com facilidade. Aqui estão alguns exemplos:

- Listas não ordenadas
- Diferentes níveis de cabeçalho
- **Negrito** e *itálico*
- [Links](https://stage.com.br)

### Código

Você pode incluir blocos de código com facilidade:

```javascript
function hello() {
  console.log("Olá, Stage!");
}
hello();
```

## Por que MDX?

O [MDX](https://mdxjs.com/) nos permite utilizar componentes React diretamente em nossos artigos Markdown, permitindo experiências ricas e interativas sem perder a simplicidade da escrita em texto puro.
